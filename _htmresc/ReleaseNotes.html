STM32CubeWB Firmware Package V1.4.0 / 06-December-2019

Main Changes


            Maintenance release.
  1. BLE:
    • Add new application under “Projects\P-NUCLEO-WB55.Nucleo\Applications\BLE\BLE_Peripheral_Lite”:
      • BLE_Peripheral_Lite: STM32WB simplified application without Sequencer, Low Power Manager and Timer Server.
  2. Zigbee:
    • Simplify the existing Zigbee OnOff Cluster applications for P-NUCLEO-WB55.Nucleo.
    • Add 2 new applications available under P-NUCLEO-WB55.USBDongle:
      • Zigbee_OnOff_Client_Router: How to use OnOff cluster as a Client on a centralized Zigbee network.
      • Zigbee_OnOff_Server_Coord: How to use OnOff cluster as a Server on a centralized Zigbee network.
  3. MAC 802.15.4:
    • MAC examples now ported on MDK-ARM and SW4STM32 IDEs.
  4. Maintenance release for HAL and LL drivers.
